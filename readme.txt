Menu custom fields plugin позволяет добавлять к пунктам меню дополнительные поля

== Description ==

Благодаря плагину Menu custom fields вы сможете слегкостью в настройках через панель администратора менять внешний вид любого меню вашего сайта.

Плаин поддерживает написание кода пхп для отображения меню с изменений структуры, а также внесение условий для вывода пунктов.

Предусмотрено добавление таких типов полей как:
	1. Text
	2. Image (all formats)
	3. Select
	4. Radio
	5. Checkbox

Templates of menu добавляются к определенным локациям меню добавленых для вашей темы.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Add your menus on page Appearance->Menus
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Menu custom fields screen to configure the plugin
